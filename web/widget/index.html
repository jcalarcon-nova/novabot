Integration Code

To integrate NovaBot into your website, add this code to your HTML:

<!-- NovaBot Widget CSS --> <link rel="stylesheet" href="https://cdn.novabot.example.com/widget.css"> <!-- NovaBot Widget Configuration --> <script> window.NOVABOT_API_ENDPOINT = 'https://api.novabot.example.com/invoke-agent'; window.NOVABOT_API_KEY = 'your-api-key-here'; // Optional window.NOVABOT_TITLE = 'Support Assistant'; window.NOVABOT_AUTO_OPEN = false; // Set to true to auto-open </script> <!-- NovaBot Widget JavaScript --> <script src="https://cdn.novabot.example.com/widget.js"></script>
API Configuration: Make sure to replace the API endpoint and key with your actual NovaBot deployment values. The widget will automatically handle authentication and session management.

JavaScript API

Control the widget programmatically using these methods:

// Open or close the widget NovaBot.open(); NovaBot.close(); // Send a message programmatically NovaBot.sendMessage("Hello, how can I configure my API?"); // Clear the chat history NovaBot.clearChat(); // Update API endpoint dynamically NovaBot.setEndpoint("https://new-api.example.com/invoke-agent"); // Destroy the widget instance NovaBot.destroy();
